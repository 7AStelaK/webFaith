RGB Значення. В HTML ви можете вказати кольори, використовуючи значення RGB.

rgb(255, 0, 0) відображається як червоний,
тому що червоний встановлений на найбільш високе значення (255)

rgb(255, 0, 123) Кожний параметр (red, green та blue)
визначає інтенсивність кольору від 0 до 255.

rgb(0, 0, 255)

rgb(60, 179, 113)

rgb(238, 130, 238)

rgb(255, 165, 0)

rgb(106, 90, 205)


Для відображення білого кольору встановіть всі параметри кольорів на 255,
(найвижчий параметр -255) як тут вказано: rgb(255, 255, 255).
Для відображення чорного кольору, встановіть всі параметри кольорів на 0 (найнижчій параметр - 0),
як тут вказано: rgb(0, 0, 0)

Відтінки сірого часто визначаються з використанням рівних значень для всіх трьох джерел світла

rgb(0, 0, 0)

rgb(60, 60, 60)

rgb(120, 120, 120)

rgb(180, 180, 180)

rgb(240, 240, 240)

rgb(255, 255, 255)

Використовуючи рівні значення для червоного, зеленого і синього, ви отримаєте різні відтінки сірого.
